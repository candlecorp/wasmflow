wick-config::v0.

## HostManifest

A manifest defines the starting state of a Wick host and network.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | format | u32 | The configuration manifest format. | Yes | | | version | string | The version of the configuration. | | | | host | HostConfig| null | Additional host configuration. | | | | network | NetworkManifest | The configuration for a Wick network. | | | | default_schematic | string| null | The default schematic to execute if none is provided. | | | +-+-+-+-+-+

## HostConfig

Host configuration options.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | allow_latest | bool | Whether or not to allow the :latest tag on remote artifacts. | | | | insecure_registries | string[] | A list of registries to connect to insecurely (over HTTP vs HTTPS). | | | | timeout | u64 | The timeout for network requests (in ms). | | | | id | string| null | The ID for this host, used to identify the host over the mesh. | | | | expose | string[] | The schematics to expose via RPC or the mesh, if any. | | | | mesh | MeshConfig| null | The mesh configuration. | | | | rpc | HttpConfig| null | Configuration for the GRPC server. | | | | http | HttpConfig| null | Configuration for the HTTP 1 server (development only). | | | +-+-+-+-+-+

## HttpConfig

Configuration for HTTP/S servers.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | enabled | bool | Enable/disable the server. | | | | port | u16| null | The port to bind to. | | | | address | string| null | The address to bind to. | | | | pem | string| null | Path to pem file for TLS. | | | | key | string| null | Path to key file for TLS. | | | | ca | string| null | Path to CA file. | | | +-+-+-+-+-+

## MeshConfig

Configuration used to connect to the mesh.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | enabled | bool | Enable/disable the mesh connection. | | | | address | string | The address of the NATS server. | | | | creds_path | string| null | The path to the NATS credsfile. | | | | token | string| null | The NATS token. | | | +-+-+-+-+-+

## NetworkManifest

A Wick network definition.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | name | string| null | The unique identifier for this Network. | | | | labels | Map<string,string> | The labels that apply to this network. | | | | triggers | EntrypointDefinition| null | The collection to use as the entrypoint when running as a standalone process. | | | | schematics | SchematicManifest[] | The links between capabilities and components. | | | | collections | CollectionDefinition[] | A list of component collections. | | | +-+-+-+-+-+

## EntrypointDefinition

A collection definition for the main entrypoint.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | reference | string | The reference/location of the collection. | | | | component | string | The component to use as the entrypoint. | | | | data | Map<string,any>| null | Data or configuration used to initialize the collection. | | | +-+-+-+-+-+

## CollectionDefinition

A collection definition.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | namespace | string | The local namespace for the collection. | | | | kind | CollectionKind | The kind/type of the collection. | | | | reference | string | The reference/location of the collection. | | | | data | Map<string,any>| null | Data or configuration used to initialize the collection. | | | +-+-+-+-+-+

## CollectionKind

Kind of collection.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | Native | unknown | Native collections included at compile-time in a Wick host. | | | | GrpcUrl | unknown | The URL for a separately managed GRPC endpoint. | | | | WaPC | unknown | A WaPC WebAssembly collection. | | | | Network | unknown | A local or remote Network definition. | | | +-+-+-+-+-+

## SchematicManifest

A definition for an individual Wick schematic.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | name | string | Schematic name. | Yes | | | collections | string[] | A list of collections the schematic uses. | | | | instances | Map<string,ComponentDefinition> | A map from component reference to its target. | | | | connections | ConnectionDefinition[] | A list of connections from component to component. | | | | constraints | Map<string,string> | A map of constraints and values that limit where this schematic can run. | | | +-+-+-+-+-+

## ComponentDefinition

A single component definition.

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | id | string | The ID of the component (i.e. the alias, key, or namespace). | Yes | | | data | Map<string,any>| null | Data to associate with the reference. | | | +-+-+-+-+-+

## ConnectionDefinition

A connection between components. This can be specified in short-form syntax (where applicable).

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | from | ConnectionTargetDefinition | The originating component from upstream. | | | | to | ConnectionTargetDefinition | The destination component (downstream). | | | +-+-+-+-+-+

## ConnectionTargetDefinition

A connection target e.g. a port on a reference. This can be specified in short-form syntax (where applicable).

+-+-+-+-+-+ | Field name | Type | Description | Required | Deprecated | | instance | string | The instance name of the referenced component. | Yes | | | port | string | The component's port. | Yes | | | data | Map<string,any>| null | Data to associate with a connection. | | | +-+-+-+-+-+
